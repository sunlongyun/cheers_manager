叮当农场管理后台

平台介绍

“叮当农场”是上海赤松信息技术限公司开发的一款农产品交易平台。

货源地直接发货，没有中间商赚差价， 顾客少花钱，农户多挣钱。

解决农产品销售问题，同时让大城市的白领可以比较低廉的价格买到最新鲜的农产品。

管理后台使用说明

* 管理员拥有 供应商管理、商品管理、订单管理的管理权限

* 管理员只能管理自己权限访问内的供应商、商品、订单

* 超级管理员拥有全部菜单，全部数据的管理权限

* 平台详细记录每一次操作记录，平台管理员需要对每一次操作负责
